# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.1] - 2024-12-30

### Added

- Initial release of AI Travel Planner with Durable Agents (Python version)
- Multi-agent workflow with Destination Recommender, Itinerary Planner, and Local Recommendations agents
- Human-in-the-loop approval workflow
- React frontend with real-time progress tracking
- Azure deployment via Azure Developer CLI (azd)
- Infrastructure as Code using Bicep with Azure Verified Modules (AVM)
- Flex Consumption hosting for Azure Functions
- Integration with Azure OpenAI (GPT-4o-mini)
- Durable Task Scheduler for reliable orchestration
